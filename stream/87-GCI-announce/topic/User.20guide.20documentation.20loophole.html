Hey GCI students! Recently @Rishi Gupta posted some new user guide documentation tasks to complete, but some of these features have been already previously documented. Some examples of this include the Deactivate/reactivate a user, Change a user's name, and the Change default language feature tasks, among at least 3 others. If you're looking for easy task points, I strongly encourage you to claim these tasks before they're completed. Good luck!
